# Leaseweb Developer API Python Wrapper
An extended python api-wrapper for the [Leaseweb Developer API](https://developer.leaseweb.com/docs/). Supports all methods and types of responses.

## Installation

Via uv

``` bash
$ uv sync
$ uv build
$ pip install -e .
```
